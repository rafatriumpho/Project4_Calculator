# Calculator with Menu (Python)

This is my fourth Python project, built during Week 4 of my Python learning journey.

The program is a terminal-based calculator that allows the user to choose operations from a menu and perform calculations interactively.

---

##  Features

- Menu-based interface

- Four operations:

  - Addition

  - Subtraction

  - Multiplication

  - Division

- Continuous operation mode (user stays inside an operation)

- Custom command to return to the main menu

- Division by zero handling

- Clean and structured code using functions

---

##  What I Learned

- How to use functions to organize code

- How to build menu-based systems

- How to use loops inside loops (nested loops)

- How to separate logic (menu vs operations)

- Input handling and validation

- Writing cleaner and more reusable code

---

##  How It Works

1. The program displays a menu with available operations

2. The user selects an operation

3. The user enters numbers to perform calculations

4. The program keeps running the operation until the user types a command (like "exit") to return to the menu
